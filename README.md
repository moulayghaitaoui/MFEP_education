# منصة التميز الرقمي - وزارة التكوين والتعليم المهنيين (MFEP)

تطبيق ويب متطور يهدف إلى رقمنة قطاع التكوين والتعليم المهنيين في الجزائر، مصمم بواجهات عصرية واحترافية تلبي تطلعات "الإدارة الرقمية الحديثة".

## 🚀 نظرة عامة
المشروع عبارة عن واجهة أمامية (Frontend Only) مبنية باستخدام **Laravel Blade** و **TailwindCSS**، مع تعزيز التفاعل باستخدام **Alpine.js**. المنصة مصممة لدعم اللغة العربية (RTL) بالكامل وتوفر تجربة مستخدم فريدة لأربعة أدوار رئيسية.

## 👥 الأدوار والوظائف المدعومة

### 1. الإدارة العليا (Super Admin)
*   **مركز التحكم الوطني**: إحصائيات شاملة على مستوى الوطن.
*   **مراقبة النظام**: متابعة حالة السيرفرات والخدمات.
*   **التقارير الوطنية**: تحليل الأداء العام للمديريات والولايات.
*   **إعدادات المنصة**: إدارة مفاتيح الأمان والتهيئة العامة.

### 2. المديرية الولائية (Regional Directorate)
*   **إدارة المؤسسات**: تسيير المعاهد والمراكز التكوينية التابعة للولاية.
*   **التسيير البيداغوجي**: إدارة الأفواج والجداول الزمنية والمواقيت.
*   **الرقابة والمتابعة**: سجلات الحضور والغياب اليومية للأساتذة والمتربصين.
*   **النتائج والتقارير**: مركز استخراج التقارير الولائية والنتائج النهائية.

### 3. فضاء الأستاذ (Teacher/Instructor)
*   **إدارة المحتوى**: رفع الدروس، الفيديوهات، والمستندات التعليمية.
*   **بناء الاختبارات**: نظام ديناميكي لإنشاء اختبارات تفاعلية.
*   **متابعة الطلبة**: تحليل تقدم الطلاب وتقييم أدائهم الأكاديمي.
*   **الجدولة الذكية**: تنظيم الحصص المباشرة والحضورية.

### 4. فضاء المتربص (Student Experience)
*   **تجربة تعلم Coursera**: واجهة مخصصة لمشاهدة الدروس ومتابعة التقدم.
*   **الاختبارات الرقمية**: إجراء التقييمات والحصول على نتائج فورية.
*   **مركز الشهادات**: تحميل الشهادات الرقمية المعتمدة والموقعة إلكترونياً.

## 🛠️ التقنيات المستخدمة
*   **Framework**: Laravel 11
*   **Styling**: TailwindCSS 3.4
*   **Interactivity**: Alpine.js (for Modals, Sidebars, Quizzes, and State Management)
*   **Icons**: Lucide Icons
*   **Fonts**: Tajawal (Google Fonts)

## 📁 هيكلية المشروع (Views Structure)
```text
resources/views/
├── admin/          # واجهات الإدارة العليا
├── direction/      # واجهات المديرية الولائية
├── teacher/        # واجهات الأستاذ والمحتوى
├── student/        # واجهات المتربص والتعلم
├── components/     # المكونات القابلة لإعادة الاستخدام (Sidebar, Navbar, etc.)
└── layouts/        # القوالب الأساسية (Conditional Layouts)
```

## ⚙️ التشغيل والمعاينة
للمعاينة المحلية، يرجى التأكد من تثبيت متطلبات Laravel ثم تشغيل الأوامر التالية:

1. تثبيت الاعتمادات:
```bash
composer install
npm install
```

2. بناء الأصول:
```bash
npm run dev # للمطوير
# أو
npm run build # للإنتاج
```

3. تشغيل الخادم:
```bash
php artisan serve
```
